# Global Banking Risk Analytics & Fraud Detection Platform

## Project Overview
Designed and implemented an end-to-end banking data engineering and analytics platform using Python, PostgreSQL, Docker, and Power BI.

## Tech Stack
- Python
- PostgreSQL
- Docker
- Power BI
- DAX
- Pandas
- SQLAlchemy

## Architecture
Kaggle Dataset → Python ETL → PostgreSQL → Analytics Layer → Power BI Dashboards

## Features
- Raw, staging, and analytics schemas
- Fraud detection workflows
- Branch KPI reporting
- Executive dashboards
- Advanced DAX KPIs
- Risk monitoring analytics
